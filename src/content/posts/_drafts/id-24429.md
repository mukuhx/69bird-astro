中国語例文を覚えるためにAnkiを使い始めました。

Ankiとは単語帳アプリです。

Ankiを使うことで、中国語の学習を通勤電車やカフェなど、隙間時間を利用して効率的に学習できています。

本記事ではAnkiで音声付きの中国語例文を覚えるためのカード作成方法について解説します。

## Ankiを選んだ理由

AnkiのiPhoneアプリは有料で、2021年7月に購入し3,060円でした。

```
AnkiMobile Flashcards開発元:Ankitects Pty Ltd¥3,060posted withアプリーチ
```

アプリとしては非常に高価です。

PC版は無料なので、まずはPC版をしばらく使ってみてからアプリを購入することを決めました。

単語帳アプリには無料のものもありますが、

以下の3点の理由でAnkiを使うことにしました。

1. カードのレイアウト作成の自由度が高い
2. 忘却曲線に従ってカードを自動で並び替えてくれる
3. 音声が出力できる

## 中国語Anki デッキの作成

言語を習得するには、音声を耳で聞いてすぐに意味が頭の中で思い浮かぶようになるのが理想だと考えています。

そこでカード表面は、中国語の文字ではなく、中国語の音声のみにしました。

デッキの作成はiPhoneでもできますが、アドオンが使えないため、PC（Mac）で行いました。

### ノートタイプを選択

Basic(裏表反転カード付き)を選択しました。

反転カードを作成することで、音声からだけでなく、文字からも意味を連想するトレーニングができます。

## Awesome TTSのインストール

Ankiで音声を出力させるにはAwesome TTSというアドオンを使用します。

TTSとはText to Speechの略で、文字の読み上げプログラムです。

下記ページにあるダウンロード用のコードを、Ankiのアドオンインストール画面に入れるとインストールできます。

[AwesomeTTS - Add speech to your flashcards - AnkiWeb](https://ankiweb.net/shared/info/1436550454)

Awesome TTSは、 OS X(Apple)、Google、Microsoftなど様々なTTS音源が使用できます。

最初は設定の簡単なAppleのTTSを使用していました。

これはMacやiPhoneに最初からインストールされているので、APIの設定なしで使用できて便利です。

しかし、使っていくうちに、声調が聞き取りづらいと感じるようになりました。

そこで音声の質の評価が高いMicrosoft AzureのTTSを使用してみたら、音声から声調が判別することができるようになりました。

また人の発声に近いため、真似しやすい音声です。

### Microsoft Azure TTSのAPI取得

![](/images/5BF8547C-C0A0-4BF1-9923-84B934F778AD_1_201_a-640x202.jpeg)

[Microsoft Azure Portal](https://portal.azure.com/)にMicrosoftアカウントでログインします。

Cognitive Servicesの音声から、音声サービスの作成を選択します。

音声サービスにわかりやすい名前を付けて管理します。

キーの管理という場所をクリックしてAPIキーを取得します。
